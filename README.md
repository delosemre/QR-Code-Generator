# QR Code Generator

- You can print in PNG and SVG.
- You can use it for website link or writing.

## Install
```
git clone https://github.com/delosemre/QR-Code-Generator.git
cd QR-Code-Generator
pip3 install -r requirements.txt
python3 qrcode.py
```
